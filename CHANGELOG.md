# Changelog

Toutes les modifications notables de ce projet seront documentées dans ce fichier.

## [1.0.0] - 2026-04-25

### ADD
- Script initial `start.py` pour l'extraction SQLite.
- Support des schémas récents de Shazam macOS (`ZTRACKNAME`, `ZSUBTITLE`, `ZDATE`).
- Détection du mode **Auto-Shazam** (`ZISAUTO`).
- Génération automatique de liens de recherche vers **Beatport** et **Google**.
- Export multi-format : **CSV** et **Markdown Dashboard**.
- Documentation de base (`README.md`, `CHANGELOG.md`).
